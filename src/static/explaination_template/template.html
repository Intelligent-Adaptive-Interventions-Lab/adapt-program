def greet(name):
    # A simple function to greet the user
    print(f"Hello, {name}!")

Explanation:

The code above defines a Python function called greet that takes a name parameter and prints a greeting message to the user with the provided name.

The function uses an f-string to format the greeting message, which is a convenient way to include variables inside a string in Python.

For example, if you call the function with greet("John"), it will print: Hello, John!
