Python Function: earlier_name

The earlier_name function takes two string parameters, name1 and name2, and returns the name that comes first alphabetically.

Parameters:

  • name1 (str): The first name.
  • name2 (str): The second name.

Returns:

str: The name that comes first alphabetically.

Examples:


        earlier_name('Jen', 'Paul')
        # Output: 'Jen'

        earlier_name('Colin', 'Colin')
        # Output: 'Colin'

Function Implementation:


        def earlier_name(name1: str, name2: str) -> str:
            if name1 < name2:
                return name1
            else:
                return name2
